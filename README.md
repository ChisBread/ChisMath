## 数据结构与内存管理(初版)
  - 一个Chis::Expr中有一个root成员变量，保存表达式AST的根节点的地址。
  而所有节点按照可读的形式保存在成员变量node_pool中，避免了内存泄漏。缺点是cache不友好，在后续开发中尝试改进。
  - 每一个表达式有两个成员 id_type与id_value，保存着表达式中的符号信息。
  - 为了保证基础库开发中效率的一致性，所有的数学函数只接受右值，并为各种形式的运算操作设计了移动构造。
